# 또박또박 — 말하면 글자로 보여 드려요

귀가 어두우신 분과 대화할 때 쓰는 **음성 → 글자** 화면입니다.
보청기가 없어도, 상대가 말한 문장을 **큰 글씨로** 화면에 띄워 어르신이 읽고 답하실 수 있게 돕습니다.

`talk/index.html` **하나로 끝**입니다. 설치·서버 없이 브라우저에서 바로 동작하고, 인터넷이 끊겨도 됩니다(글씨체만 온라인일 때 더 예쁘게 나옴).

## 어떻게 쓰나요

1. 어르신께 휴대폰을 드리거나, 잘 보이게 둡니다.
2. 가운데 아래 **🎤 파란 버튼**을 누릅니다. → "듣고 있어요" 표시.
3. 상대가 **또박또박** 말합니다. 말하는 동안 화면에 글씨가 실시간으로 올라옵니다(흐린 글씨 = 아직 정리 중).
4. 말을 다 하면 **빨간 "말 다 했어요" 버튼**을 누릅니다. → 문장이 완성되어 **큰 글씨로** 고정됩니다.
5. 어르신이 읽고 대답하십니다. 다음 말을 할 땐 다시 버튼을 누르면 됩니다.

- **글씨 크기**: 오른쪽 위 `가−` `가＋` 로 키우고 줄일 수 있어요(기억됩니다).
- **화면 지우기**: 문장을 지우고 처음으로 돌아갑니다.

## 권장 환경

- **안드로이드 / 컴퓨터: 크롬(Chrome)**
- **아이폰: 사파리(Safari)**
- 처음 누르면 **마이크 사용 허용**을 물어봅니다 → **허용**을 눌러야 동작합니다.
- 한국어(`ko-KR`) 인식을 사용합니다. 음성 인식은 브라우저 기능이라 기기·환경에 따라 정확도가 다를 수 있어요.

> 음성 인식 처리 방식은 브라우저(기기)가 정합니다. 일부 브라우저는 인식을 위해 음성을 자체 서버로 보낼 수 있으니, 민감한 대화에는 주의하세요. 이 앱 자체는 어떤 서버로도 내용을 보내지 않습니다.

## 배포 (GitHub Pages)

이 저장소가 GitHub Pages로 배포돼 있으면, 아래 주소로 바로 열 수 있습니다:

```
https://{계정명}.github.io/voc-dashboard/talk/
```

> 음성 인식과 마이크는 보안상 **https** 또는 **localhost**에서만 동작합니다.
> GitHub Pages 주소(https)에서는 정상 동작하고, `file://`로 열면 마이크가 막힐 수 있어요.

휴대폰 홈 화면에 추가해 두면(브라우저 메뉴 → "홈 화면에 추가") 앱처럼 한 번에 열 수 있습니다.
